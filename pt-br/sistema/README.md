## Scripts de verificação – Linux

Esta seção documenta os scripts localizados em:

scripts/linux/

Eles foram criados com foco em segurança defensiva, aprendizado e consciência digital.

# basic_security_check.sh

# Objetivo:
Realizar verificações básicas no sistema operacional para identificar sinais comuns de risco.

# O que ele verifica:

permissões suspeitas

processos incomuns

variáveis de ambiente relevantes

diretórios sensíveis

# O que ele NÃO faz:

não acessa a internet

não coleta dados

não interage com jogos

não altera o sistema

# Quando usar:

antes de jogar

após instalar mods

após instalar launchers ou ferramentas externas

# minecraft_mod_check.sh

# Objetivo:
Ajudar o usuário a ter consciência sobre mods presentes no sistema.

# O que ele faz:

verifica pastas comuns de mods

alerta sobre nomes ou padrões incomuns

ajuda o usuário a revisar o ambiente

# Importante:
Este script não afirma que um mod é malicioso.
Ele apenas incentiva a revisão manual e consciente.

# Aviso
Estes scripts são educacionais.
Eles não substituem ferramentas profissionais de segurança.


## Scripts de verificação – Windows

Localização:
scripts/windows/


# basic_security_check.ps1

**Objetivo:**  
Executar verificações básicas de segurança no Windows.

**O que verifica:**
- processos em execução
- permissões básicas
- variáveis de ambiente
- diretórios comuns de risco

**O que NÃO faz:**
- não envia dados
- não acessa servidores
- não altera configurações
- não interage com o jogo


# minecraft_mod_check.ps1

**Objetivo:**  
Ajudar na revisão consciente de mods instalados.

**O que faz:**
- verifica pastas comuns de mods
- alerta nomes incomuns
- incentiva inspeção manual

**Aviso:**  
Não afirma que algo é malicioso.  
É apenas ferramenta educativa.
