# Player Security Checklist

This checklist is meant to help Minecraft players maintain basic digital hygiene.


## Before installing mods

- Verify the source of the mod
- Avoid unknown download sites
- Prefer well-known communities
- Scan files with antivirus if available


## System hygiene

- Keep the operating system updated
- Avoid running unknown executables
- Review startup programs
- Use strong passwords


## While playing

- Avoid suspicious links in chat
- Do not share account credentials
- Be cautious with “free coin” promises


## After installing tools or launchers

- Run local security checks
- Review file permissions
- Remove unused software


## Mindset

Security is about awareness, not fear.
